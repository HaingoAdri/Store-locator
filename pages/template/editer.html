Éditeur Visuel

Utilisez l'éditeur visuel pour personnaliser l'apparence de votre store locator. Modifiez les couleurs, les polices, les styles et les pins de localisation directement dans l'éditeur.

Filtres et Prévisualisation en Direct
Modifier Filtres

Ajustez les options de filtre pour que les utilisateurs puissent trouver des emplacements spécifiques plus facilement.

Prévisualisation

Avant de sauvegarder vos modifications, prévisualisez le design avec les pins, les couleurs et les polices modifiées.

Sauvegarde

Enregistrez vos modifications (pins, filtres, couleurs, polices) pour les appliquer à votre store locator. Vous pouvez également sauvegarder les configurations sous différents noms pour une utilisation future.
