Instructions pour Ajouter le Store Locator

Instructions: Copiez et collez le code dans un bloc HTML de la page où vous souhaitez que le store locator apparaisse.

Option 1 – Toutes les Localisations
<div id='storerocket-widget' style='width:100%;' data-storerocket-env='p' data-storerocket-id='Ax86Nka8y1'>
    <p style='text-align:center;font-size:13px;padding:10px;'>Store locator is loading from StoreRocket 
    <a target='_blank' href='https://storerocket.io' style='font-size:13px;'>Store Locator App</a>..
    </p>
</div>
<script>
    (function(){
        var a=document.createElement('script');
        a.type='text/javascript';
        a.async=!0;
        a.src='https://cdn.storerocket.io/js/widget-mb.js';
        var b=document.getElementsByTagName('script')[0];
        b.parentNode.insertBefore(a,b);
    }());
</script>

Options (ajoutez les attributs à l'élément #storerocket-widget) :

  • data-storerocket-lang (optionnel; exemple: en) - force l'utilisation d'une langue spécifique.
  • data-storerocket-location (optionnel; exemple: location-slug) - force le store locator à ouvrir une localisation spécifique.
  • data-storerocket-enabled-filters (optionnel; exemple: 1,2,3,4) - force l'affichage de filtres spécifiques.
  • data-storerocket-filters (optionnel; exemple: 1,2,3,4) - charge le store locator avec des filtres spécifiques pré-sélectionnés.
  • data-storerocket-geolocation (optionnel; exemple: false) - force l'activation ou la désactivation de la géolocalisation.
  • data-storerocket-mapbox-style (optionnel; seulement pour Mapbox; exemple: mapbox://styles/mapbox-user/style-id) - définit dynamiquement le style Mapbox pour la carte.
Option 2 – Localisations Pré-filtrées

Vous devez d'abord créer des filtres. Si vous souhaitez que le store locator affiche uniquement les localisations de filtres spécifiques, sélectionnez les filtres ci-dessous et copiez le code.

<div id='storerocket-widget' style='width:100%;' data-storerocket-env='p' data-storerocket-id='Ax86Nka8y1'>
    <p style='text-align:center;font-size:13px;padding:10px;'>Store locator is loading from StoreRocket 
    <a target='_blank' href='https://storerocket.io' style='font-size:13px;'>Store Locator App</a>..
    </p>
</div>
<script>
    (function(){
        var a=document.createElement('script');
        a.type='text/javascript';
        a.async=!0;
        a.src='https://cdn.storerocket.io/js/widget-mb.js';
        var b=document.getElementsByTagName('script')[0];
        b.parentNode.insertBefore(a,b);
    }());
</script>
Option 3 – Initialisation du Widget Programmatiquement

Pour les applications à page unique qui rechargent le contenu dynamiquement, vous pouvez initialiser le widget dans votre code JavaScript.

<div class="storerocket-store-locator"></div>

<script src="//cdn.storerocket.io/widget.js"></script>
<script>
    StoreRocket.init({
        selector: ".storerocket-store-locator",
        account: "Ax86Nka8y1"
    });
</script>

Options :

  • selector (requis) - Sélecteur CSS où injecter le store locator.
  • account (requis) - Votre ID de compte.
  • location (optionnel; exemple: location-slug) - force le store locator à ouvrir une localisation spécifique.
  • tracking (optionnel; par défaut: true) - activer/désactiver le suivi des événements.
  • jquery (optionnel; par défaut: true) - activer/désactiver le chargement de la version de jQuery du widget.
  • lang (optionnel; exemple: en) - force l'utilisation d'une langue spécifique.
  • filters (optionnel; chaîne; exemple: "100,101") - pré-sélectionne des filtres de recherche.

Événements :

  • storerocket-storelocator-load - Déclenché lorsque le widget du store locator se charge.
  • storerocket-storelocator-search (paramètre: query) - Déclenché lorsqu'un utilisateur effectue une recherche.
